# mattermost-plugin-disallow-dm

Mattermost server plugin that blocks direct (DM) and group (GM) messages between users.

## Working conventions

- **All code and project documentation MUST be written in English** (source, comments, specs, README, commit messages, this file).
- **Reply to the user in the language they asked in** (e.g. Russian question → Russian answer). This applies only to conversational replies, not to committed artifacts.
- **Ask clarifying questions as multiple-choice polls (`/poll`)**, i.e. always offer answer options via the interactive question panel rather than free-form prose questions.
